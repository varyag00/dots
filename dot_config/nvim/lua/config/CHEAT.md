# LazyVim Cheatsheet

## Keymaps

```lua
-- NOTE: terminal
<C-/>         -- open/close terminal

-- NOTE: help
<leader>sh    -- open lazyvim help
<leader>sk    -- keybind help

-- NOTE: navigation
<C-n> & <C-p> -- next and prev in list, insert mode
```
